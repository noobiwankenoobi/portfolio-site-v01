Tic Tac Toe in the Browser

Project Description:

Created tic tac toe in the browser using Javascript
Users can play against each other, game keeps track of wins for each player in the session
JS was continuously refactored for increased logical simplicity and readability

What's coming in Version 02 and 03:

Users can sign in, back-end keeps track of wins/losses and preferences
Users can choose an "icon" from a list to play with if they want to, instead of X's and O's
Allow user to play another user on another device in the same session
Allow user to play "against the computer", which chooses an open spot on the board at random
Train a machine-learning ai to win at tic-tac-toe, allow user to play against the "smart" ai if they choose
